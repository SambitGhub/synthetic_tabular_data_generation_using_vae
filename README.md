# Synthetic Tabular Data Generation Using VAE

This project explores synthetic tabular data generation using a Variational Autoencoder (VAE) on the California Housing dataset.

## Project Structure

```text
.
├── data/
│   └── california_housing.csv
├── notebooks/
│   └── synthetic_data_generation_using_vae.ipynb
├── pyproject.toml
├── uv.lock
└── README.md
```

## Setup

Clone the repository:

```bash
git clone <repository-url>
cd synthetic_tabular_data_generation_using_vae
```

Install dependencies:

```bash
uv sync
```

## Running the Project

Open the notebook:

```bash
jupyter notebook
```

and run:

```text
notebooks/synthetic_data_generation_using_vae.ipynb
```

## Dataset

California Housing Dataset

## Objective

Generate high-quality synthetic tabular data using Variational Autoencoders and evaluate the generated samples using statistical metrics.
